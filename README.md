# 🌌 Solar System Explorer

> ✨ *A modern and visually engaging journey through our Solar System*

---

## 🚀 About

**Solar System Explorer** is a beautifully designed static web project built using **pure HTML & CSS**.
It offers a clean and interactive way to explore planets, understand key space facts, and experience a well-structured multi-page website.

---

## ✨ Features

* 🪐 Explore all major planets of the Solar System
* 🌍 Dedicated pages for each planet
* 📊 Informative sections (distance, diameter, moons)
* 🎥 Embedded educational video content
* 🧠 Quiz interface design (static layout)
* 🎨 Modern space-themed UI
* ⚡ Lightweight & fast (no JavaScript used)

---

## 🛠️ Tech Stack

* HTML5
* CSS3

## ▶️ Getting Started

```bash
git clone https://github.com/your-username/solar-system-explorer.git
cd solar-system-explorer
```

Open `index.html` in your browser
or run using **Live Server** in VS Code.
